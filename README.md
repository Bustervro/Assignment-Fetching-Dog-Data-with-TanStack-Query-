# Dog API TanStack Query App

This React app uses TanStack Query to fetch and display dog data from the Dog API.

## Features

- Fetches dog breeds
- Displays loading, error, and success states
- Shows detailed breed information
- Fetches dog facts
- Fetches dog groups
- Uses TanStack Query for async data management and caching

## How to Run

```bash
npm install
npm run dev
